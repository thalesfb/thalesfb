## 📌 Featured Projects

Repositories that best represent my work:

### **ccw — Personalized Math Learning (Undergraduate Thesis)**
Academic project focused on educational data analysis, including an automated systematic review pipeline (PRISMA) and an experimental workflow for identifying student competencies. The `research/` module provides a CLI for literature retrieval and screening, report generation (CSV/HTML/BibTeX), tests, and technical documentation.  
`Python | Research Automation | Data Analysis | CLI`

### **library — Library Management System (POO2)**
Backend application developed in Java (Spring Boot) to manage authors, books, users and loan workflows. Implements Spring Data JPA/Hibernate, MySQL, validation layers and the Observer pattern for email notifications. Includes modular structure and Maven execution instructions.  
`Java | Spring Boot | JPA/Hibernate | MySQL`

### **machine_learning — ML & DL Experiments**
Collection of notebooks and organized experiments using traditional ML and deep learning (ANN, CNN, RNN, PINNs, SVM, XGBoost). Includes training pipelines, preprocessing, model evaluation and reproducible environments (`requirements.txt` and `environment.yml`).  
`Python | scikit-learn | TensorFlow | ML/DL`

### **infra_web — Infrastructure and Web Services**
Automation, provisioning, server configuration, NGINX setups, deployment standards and containerized services for DevOps practices.  
`Shell | Linux | NGINX | Docker | DevOps`

### **DevWeb — Web Development Projects**
A set of backend and frontend exercises and small applications (APIs, real-time communication with WebSocket, Dockerized services and prototypes).  
`JavaScript | Python | Docker | WebSocket`

### **Data-Structure & Seminarios-DS**
Academic repositories written in C, featuring classic and advanced data structures (e.g., red-black tree) and algorithmic implementations such as external sorting. Demonstrates strong theoretical grounding and low-level programming.  
`C | Algorithms | Data Structures`

### **SistemasDigitais**
Persistence of Vision (POV) display implemented in C++ for a Digital Systems course, exploring timing control, LED sequencing and embedded logic.  
`C++ | Embedded Systems | Digital Electronics`
